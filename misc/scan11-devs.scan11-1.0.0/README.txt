How to install it without building tools (The easy way)
Move or copy the entire scan11-devs.scan11-1.0.0 folder into your VS Code extensions directory:
Windows: %USERPROFILE%\.vscode\extensions\
macOS / Linux: ~/.vscode/extensions/
Restart Visual Studio Code.
Open a .scan11 or .s11 file.
Press Ctrl+K Ctrl+T (or Cmd+K Cmd+T on Mac), type SCAN-11 Nord and hit Enter to select your new matching theme.
